#[must_use = "Producer polling thread will stop immediately if unused"]

pub struct FutureProducer<C: Context + 'static> { /* fields omitted */ }

A producer with an internal running thread. This producer doesn't need to be polled. It can be cheaply cloned to get a reference to the same underlying producer. The internal thread can be terminated with the stop method or moving the FutureProducer out of scope.
